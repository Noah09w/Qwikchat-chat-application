# QwikChat – Real-Time Messaging Application

QwikChat is a modern real-time messaging web application designed to provide fast, secure, and seamless communication between users. The platform supports instant messaging, group chats, user authentication, and real-time updates with a clean and responsive interface.

## 🚀 Features

### Core Messaging
- Real-time messaging
- One-to-one chat
- Group chat support
- Instant message delivery
- Message timestamps

### User Features
- Secure user authentication
- User profiles
- Online/offline status
- Typing indicators
- Message read receipts

### Chat Features
- Send text messages
- Image and file sharing
- Message history
- Chat search functionality
- Message notifications

### UI / UX
- Clean modern chat interface
- Mobile responsive design
- Dark mode ready
- Smooth real-time updates

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- TypeScript
- Tailwind CSS

### Backend
- Supabase
  - Authentication
  - PostgreSQL Database
  - Realtime subscriptions
  - File storage

### Deployment
- Vercel / Netlify
